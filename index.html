How to Auto-Sync from Drexel Learn

  1. Drag this link to your bookmarks bar: Sync Drexel Learn
  2. Log into Drexel Learn
  3. Navigate to your course's Assignments or Calendar page
  4. Click the "Sync Drexel Learn" bookmark
  5. Your tasks will auto-import into this dashboard!

Quick Navigation Tips for Drexel Learn

  • Assignments: Course Home → Activities → Assignments
  • Quizzes: Course Home → Activities → Quizzes
  • Discussions: Course Home → Activities → Discussions
  • Calendar: Click the calendar icon in the top navbar
  • Grades: Course Home → Progress → Grades
  • Content: Course Home → Content (left sidebar)
